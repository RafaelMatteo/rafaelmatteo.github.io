Lenguajes y Herramientas

  • Lenguajes de Programación:
    • Python: Desarrollo backend y scripting
    • JavaScript: Desarrollo frontend, interactividad y lógica del cliente
    • HTML: Estructuración de contenido web
    • CSS: Estilización y diseño web
  • Frameworks y Bibliotecas:
    • Django: Desarrollo de aplicaciones web robustas y seguras en Python
    • Bootstrap: Framework CSS para diseño responsivo y componentes UI
    • AJAX: Técnicas de desarrollo web para crear aplicaciones web interactivas y dinámicas
  • Bases de Datos y Lenguajes de Consulta:
    • SQL (Structured Query Language): Lenguaje estándar para la gestión y manipulación de bases de datos relacionales. Utilizado para realizar consultas, actualizaciones, eliminación de datos, y la gestión de esquemas de bases de datos
    • PostgreSQL: Sistema de gestión de bases de datos objeto-relacional, conocido por su robustez y soporte avanzado de SQL
    • MySQL: Sistema de gestión de bases de datos relacional, popular por su rendimiento y facilidad de uso
    • SQLite3: Base de datos ligera para almacenamiento local en aplicaciones
  • Herramientas de Análisis de Datos:
    • Pandas: Manipulación y análisis de datos con estructuras como DataFrame y Series
    • NumPy: Computación numérica y operaciones con arrays multidimensionales
    • Matplotlib: Visualización de datos con gráficos y figuras 2D
    • Seaborn: Visualización de datos estadísticos con gráficos atractivos
    • SciPy: Computación científica y técnicas avanzadas de análisis de datos
    • GeoPandas: Extensión de Pandas para la manipulación y análisis de datos geoespaciales
  • Herramientas de Ciencia de Datos y Machine Learning:
    • Scikit-learn: Biblioteca para aprendizaje automático con herramientas para análisis de datos y minería de datos
    • TensorFlow: Plataforma de código abierto para el aprendizaje automático
    • Keras: API de alto nivel para construir y entrenar modelos de aprendizaje profundo
  • Entornos y Herramientas de Desarrollo:
    • VS Code: Entorno de desarrollo integrado (IDE) para programación
    • Anaconda: Distribución de Python y R para la ciencia de datos y aprendizaje automático
    • Jupyter: Entorno interactivo para crear y compartir documentos que contienen código, ecuaciones, visualizaciones y texto narrativo
    • JupyterLab: Entorno de desarrollo interactivo basado en la web para notebooks, código y datos
  • Otras Herramientas:
    • Bash: Intérprete de comandos de Unix, utilizado para scripting y administración del sistema
    • Git: Control de versiones y colaboración en código
