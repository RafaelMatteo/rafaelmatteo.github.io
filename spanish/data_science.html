Ciencia de Datos...

En esta sección, presento mis proyectos en el campo de la ciencia de datos. Utilizando técnicas avanzadas de modelado y análisis, como la regresión, clasificación y clustering, he abordado problemas complejos y generado soluciones innovadoras.
Los proyectos que verás aquí demuestran mi capacidad para manejar datasets grandes y diversos, realizar análisis exploratorios profundos y construir modelos predictivos robustos.

  • SQL y Python
    • Datos de la Ciudad de Chicago
      Boletas de calificaciones de progreso (2011-2012):
      Este conjunto de datos muestra todos los datos de rendimiento a nivel escolar utilizados para crear los Boletines de Calificaciones Escolares de BCE para el año escolar 2011-2012.
      • Conjunto de datos extraido de: Chicago Public Schools

      Datos de indicadores Socioeconómico (2008-2012):
      Este conjunto de datos contiene una selección de seis indicadores socioeconómicos de importancia para la salud pública y un “índice de dificultades” para cada área comunitaria de Chicago, para los años 2008 a 2012.
      • Conjunto de datos extraido de: Chicago Census Data

      Datos de Crimenes (2012):
      Este conjunto de datos refleja los incidentes delictivos denunciados (con excepción de los asesinatos, donde existen datos para cada víctima) que ocurrieron en la ciudad de Chicago en el año 2012.
      Estos conjuntos de datos son proporcionados por el portal de datos de la Ciudad de Chicago.
      • Conjunto de datos extraido de: Chicago Crime Data

    • Herramientas utilizadas: Python, Jupyter Lab, Pandas, SQLite3, SQLalchemy, Matplotlib, Seaborn

  • Web Scraping
    • HTML Color Codes
      En este proyecto, he desarrollado una solución de web scraping para extraer información detallada sobre códigos de color HTML desde un sitio web especializado. Utilizando técnicas avanzadas de scraping, mi herramienta recoge datos precisos sobre una amplia gama de colores, incluyendo sus nombres, códigos hexadecimales y representaciones RGB.
      Proceso: La solución navega por el sitio web objetivo, identifica y extrae información relevante sobre los colores, y organiza los datos en un formato estructurado para su posterior análisis o uso.
    • Conjunto de datos extraido de: Color Codes
    • Herramientas utilizadas: Python, Jupyter Lab, Pandas, Request, BS4 (BeautifulSoup)
    • Puedes descargar el proyecto completo (zip) o solo el código (pdf).
      HTMLColorCodes.zip HTMLColorCodes.pdf
  • Web Scraping
    • Poblacion Mundial
      En este proyecto, he desarrollado una solución de web scraping para extraer datos detallados sobre la población mundial desde el sitio web Worldometers. Mi herramienta está diseñada para recopilar información actualizada y precisa sobre la población de diferentes países y regiones, proporcionando una visión integral de las estadísticas globales.
      Proceso: La herramienta navega por Worldometers, identifica las tablas y secciones relevantes, y extrae datos sobre la población de países y regiones en tiempo real. Los datos se organizan y almacenan en un formato estructurado para su análisis posterior.
    • Conjunto de datos extraido de: World Population
    • Herramientas utilizadas: Python, Jupyter Lab, Pandas, Request, BS4 (BeautifulSoup)
