  • SQL and Python
    • Chicago City Data
      Progress Report Cards (2011-2012):
      This dataset displays all school-level performance data used to create the SRC School Report Cards for the 2011-2012 school year.
      • Dataset extracted from: Chicago Public Schools

      Socioeconomic indicator data (2008-2012):
      This data set contains a selection of six socioeconomic indicators of importance to public health and a “hardship index” for each community area in Chicago, for the years 2008 to 2012.
      • Dataset extracted from: Chicago Census Data

      Crime Data (2012):
      This dataset reflects reported crime incidents (with the exception of murders, where data exists for each victim) that occurred in the City of Chicago in 2012.
      These datasets are provided by the City of Chicago's data portal.
      • Dataset extracted from: Chicago Crime Data

    • Tools: Python, Jupyter Lab, Pandas, SQLite3, SQLalchemy, Matplotlib, Seaborn

  • Web Scraping
    • World Population
      In this project, I have developed a web scraping solution to extract detailed data on the world's population from the Worldometers website. My tool is designed to collect up-to-date and accurate information on the population of different countries and regions, providing a comprehensive view of global statistics.
      Process: The tool browses Worldometers, identifies the relevant tables and sections, and extracts data on the population of countries and regions in real-time. The data is organized and stored in a structured format for further analysis.
