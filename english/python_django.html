  • Django
    • During my Django learning, I have gained skills in implementing robust and functional web applications.
      A prominent example is the development of a School Management Administration System.
      In this project, I implemented a CRUD application that manages users, grades, courses, students and their respective family members. I also developed specific logic for assigning monthly grades.
      Initially, I used PostgreSQL as the database, but later migrated to SQLite to facilitate better distribution and management of the system.
      This project has allowed me to delve deeper into key Django concepts, such as creating models, views and templates, as well as database management and implementing user authentication and authorization.
      This learning process has been very enriching and has provided me with a solid foundation for future developments in Django.
      • Tools: Python - Django - SQLite3 - HTML - JavaScript - AJAX - Bootstrap - CSS
